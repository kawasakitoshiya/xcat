# xcat

Expanded cat tool especially for Excel

This project is on WIP.

# Description
`xcat` is tool to show excel and csv file simultaneously on command line interface.

``` abap
$ xcat price_list.xlsx
#,name,price
1,pencil,110
2,pen,150
3,apple,200
4,banana,100

$ xcat price_list.csv
1,pencil,110
2,pen,150
3,apple,200
4,banana,100



```
